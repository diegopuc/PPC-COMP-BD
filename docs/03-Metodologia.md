
As alterações no projeto são registradas por meio de **commits**, permitindo acompanhar a evolução da modelagem e da implementação do banco de dados.

# Organização do Projeto

As atividades do projeto foram organizadas de acordo com as etapas de construção do banco de dados.

| Etapa | Descrição |
|------|------|
| Levantamento do domínio | Identificação das informações do problema |
| Modelagem conceitual | Construção do MER |
| Modelagem lógica | Conversão para modelo relacional |
| Modelagem física | Definição das tabelas e tipos de dados |
| Implementação | Criação do banco de dados em SQL |
| Consultas | Desenvolvimento de consultas SQL |

Cada etapa depende da etapa anterior, garantindo consistência entre o domínio do problema e a estrutura final do banco de dados.

---

# Divisão de Atividades

As atividades do projeto foram distribuídas entre os membros do grupo.

| Atividade | Responsável |
|------|------|
| Levantamento das informações do domínio | Equipe |
| Construção do MER | Integrante 1 |
| Definição do modelo lógico | Integrante 2 |
| Implementação do banco de dados | Integrante 3 |
| Desenvolvimento das consultas SQL | Integrante 4 |
| Documentação do projeto | Equipe |

Essa divisão permitiu que cada integrante contribuísse em diferentes etapas do desenvolvimento do banco de dados.

# Ferramentas Utilizadas

As principais ferramentas utilizadas no projeto foram:

**brModelo ou draw.io**

Ferramenta utilizada para criação do Modelo Entidade-Relacionamento (MER), permitindo representar graficamente entidades, atributos e relacionamentos.

**MySQL / PostgreSQL**

Sistema Gerenciador de Banco de Dados utilizado para implementação do modelo físico e armazenamento das informações.

**MySQL Workbench / DBeaver**

Ferramentas utilizadas para executar comandos SQL, criar tabelas e testar consultas.

**GitHub**

Utilizado para armazenar o repositório do projeto, permitindo versionamento da documentação e dos scripts SQL.

**Visual Studio Code**

Editor de código utilizado para escrita e organização dos scripts SQL e da documentação do projeto.
